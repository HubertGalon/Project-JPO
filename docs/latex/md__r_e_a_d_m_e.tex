Aplikacja do monitorowania jakości powietrza w Polsce wykorzystująca dane z API Głównego Inspektoratu Ochrony Środowiska.\hypertarget{md__r_e_a_d_m_e_autotoc_md1}{}\doxysection{\texorpdfstring{Opis projektu}{Opis projektu}}\label{md__r_e_a_d_m_e_autotoc_md1}
Aplikacja umożliwia\+:
\begin{DoxyItemize}
\item Pobieranie aktualnych danych o stacjach pomiarowych, czujnikach i pomiarach
\item Wyszukiwanie stacji po mieście lub lokalizacji geograficznej
\item Wyświetlanie danych pomiarowych w formie wykresów
\item Analizowanie trendów w danych
\item Pracę w trybie offline z wykorzystaniem zapisanych danych
\end{DoxyItemize}\hypertarget{md__r_e_a_d_m_e_autotoc_md2}{}\doxysection{\texorpdfstring{Wymagania}{Wymagania}}\label{md__r_e_a_d_m_e_autotoc_md2}

\begin{DoxyItemize}

\item Kompilator C++ zgodny ze standardem C++14 lub nowszym
\item Połączenie internetowe (opcjonalne, aplikacja może działać offline)
\item Uprawnienia do zapisu w katalogu aplikacji (dla lokalnej bazy danych)
\end{DoxyItemize}\hypertarget{md__r_e_a_d_m_e_autotoc_md3}{}\doxysection{\texorpdfstring{Struktura projektu}{Struktura projektu}}\label{md__r_e_a_d_m_e_autotoc_md3}
